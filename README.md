# CS-332: Lab-1

## Purpose:
To read a given file and count the number of times given keywords appear in the file.

## How to Compile:
'gcc hw1.c -o hw1'

## How to Execute:
'./hw1 [keyword(s)] < [input file name]'
    Note: multiple keywords should be delimited by a space character (" ").
